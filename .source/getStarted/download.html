你可以很方便地从Github上下载项目源码，master分支上保存着最新版的D2Server，dev分支则汇集着正在开发的新功能。

通过Git Clone（推荐）

正在对D2Server的下载解决方案进行优化，Git方式暂时无法使用，敬请谅解

推荐使用git进行项目的安装，方便以后的版本迭代更新。 

git clone https://github.com/Bacra/node-d2server
cd node-d2server
git submodule update --init
更优的处理方案

由于使用项目时必须修改conf.js文件，所以，你可以先Fork一份副本，然后再通过上面的方式将Fork的副本部署到本地。

以后版本迭代更加方便，连conf.js文件都不需要顾虑，直接git pull拉取线上资源即可。

直接下载

  1. 通过这个连接，直接将项目打包成zip下载到本地。
  2. 解压文件，再下载EJS（定制版）并解压到lib/module/parseHTML/ejs目录

你也可以通过Releases获取项目之前的版本。

注：从Releases下载的源码，也需要手动下载解压EJS（定制版）。
